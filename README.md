#  Currency Converter & Price Scraper

## Description
This project scrapes book prices from Books To Scrape, converts them into another currency using a live API, and saves results in CSV and JSON formats.



## Features
- Web scraping using BeautifulSoup
- Currency conversion using live API
- Data storage (CSV + JSON)
- Error handling
- Clean tabular display



## Output
Files are saved in `/output` folder:
- products.csv
- products.json
